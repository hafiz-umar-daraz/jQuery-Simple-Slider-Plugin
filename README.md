# LinkIt jQuery Plugin

#### linkIt is an easy-to-use plugin that allows you to link any element 
# Version
#### 0.2.0

# Usage
    $('span').linkIt({
    	href : 'https://test.com',
    	text : 'Click Me',
    	target:'_blank'
    });
